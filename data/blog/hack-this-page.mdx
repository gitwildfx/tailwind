## **Step 2: Set Up the Repository on GitHub**

After deploying the template, connect it to a GitHub repository for version control.

**Steps:**
1. Go to your project on Vercel.
2. Click on **Settings** and select **Git Integration**.
3. Connect your GitHub account and create a new repository.
4. Push the template code to the GitHub repository.

## **Step 4: Deploy Updates to Vercel**

With every push to the main branch of your GitHub repository, Vercel automatically deploys your changes.

**Steps:**
1. Commit and push your changes.  
   ```bash
   git add .
   git commit -m "Customize blog"
   git push origin main
   ```
2. Vercel will trigger a new deployment, and you can view the updated blog at your live URL.
